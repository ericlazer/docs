## Text Formatting

Regular text looks like this.

**Bold text** and *italic text* are supported.

***Bold and italic*** can be combined.

~~Strikethrough text~~ is also available.

## Lists

- Second item
  - Nested item
  - Another nested item

### Ordered List
1. First step
2. Second step
   1. Sub-step A
   2. Sub-step B
3. Third step

---

  This is a note callout

</Warning>

  This is a tip callout
</Tip>

## Accordions

<AccordionGroup>
  <Accordion title="First Section">
    This is the content of the first accordion.
  </Accordion>
  
  <Accordion title="Second Section">
    This is the content of the second accordion.

    - Lists
    - Multiple paragraphs
    - Even code blocks
  </Accordion>
  
  <Accordion title="Third Section">
    This is the third section's content.
  </Accordion>

## Images

{/* Local images from your assets folder */}
<img src="/images/blog_1.png" alt="Example image" />

{/* Remote images */}
<img
  src="/images/blog_1.png"
  alt="Remote image"
  width="300"
  height="200"
/>

{/* Frame component for better image presentation */}
<Frame>
  <img src="/images/blog_1.png" alt="Screenshot with frame" />
</Frame>

{/* Caption for images */}
<Frame caption="This is a caption for the image">
  <img src="/images/blog_1.png" alt="Image with caption" />
</Frame>
